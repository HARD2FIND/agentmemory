# AgentMemory Viewer

Ce dossier contient les scripts pour lancer le viewer AgentMemory et configurer les dashboards.

## Lancer le viewer

```bash
npm run viewer
```

Le viewer est accessible sur `http://localhost:3113` par défaut.

## Fonctionnalités

- Recherche sémantique dans la mémoire par mots-clés ou tags.
- Replay des sessions passées.
- Visualisation des graphes de connaissances.
- Métriques de précision (Recall@5) et de consommation de tokens.
