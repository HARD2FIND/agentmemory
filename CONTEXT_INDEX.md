# Index de Contexte

Cet index mappe les documents et notes disponibles dans la base de connaissances (Obsidian Vault et AgentMemory) pour faciliter la recherche par les agents.

| Titre | Chemin | Tags | Statut | Poids/Importance |
|---|---|---|---|---|
| Architecture Agentic OS | `ARCHITECTURE.md` | `#architecture`, `#system` | Actif | Élevé |
| Sécurité et Guardrails | `SECURITY_AND_GUARDRAILS.md` | `#security`, `#policy` | Actif | Élevé |
| Manuel Opératoire | `OPERATING_MANUAL.md` | `#ops`, `#manual` | Actif | Moyen |

*(Cet index doit être mis à jour automatiquement par le workflow n8n de synchronisation de la mémoire)*

## Mise à jour manuelle

Si vous ajoutez un document manuellement et souhaitez qu'il soit immédiatement disponible pour les agents :
1. Ajoutez une entrée dans le tableau ci-dessus.
2. Assurez-vous que le chemin est correct par rapport à la racine du projet ou au Vault Obsidian.
3. Utilisez des tags pertinents pour faciliter la recherche sémantique.
